# Automatisation AI — Landing page BTP

Landing page one-page pour une activité de consultant en automatisation IA destinée aux artisans et PME du BTP.

## Contenu

- `index.html` — page complète (HTML + CSS + JS inline, sans framework, Google Fonts uniquement).

## Aperçu local

Ouvrez simplement `index.html` dans un navigateur, ou servez le dossier :

```bash
python3 -m http.server 8000
```

Puis rendez-vous sur http://localhost:8000

## À personnaliser avant mise en ligne

- Nom de marque (placeholder : « AtelierAuto »).
- Email et téléphone de contact.
- Bloc mentions légales (SIRET, hébergeur, etc.).
- Brancher le formulaire de contact sur un backend / CRM (voir le `// TODO backend` dans `index.html`).
